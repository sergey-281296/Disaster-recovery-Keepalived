 Почему нет скриншота из браузера:
В тестовой среде обе виртуальные машины запущены на **одном физическом хосте**.  
Из-за технических ограничений у них **одинаковые MAC-адреса**, поэтому:
- ARP-запросы работают некорректно
- Виртуальный IP доступен только **внутри сети виртуальных машин**
- Из внешней сети (браузера) доступ к VIP невозможен

**В реальной инфраструктуре** с разными физическими серверами:
- Виртуальный IP был бы доступен из браузера
- Принцип работы остаётся **точно таким же**
